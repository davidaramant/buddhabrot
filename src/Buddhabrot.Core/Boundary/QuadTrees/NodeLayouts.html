Notes

VisitedRegions and RegionLookup similar but distinct node formats. The two types of tree have some similarities and differences:

  • RegionLookup is deduplicated (subtrees can be referenced multiple times). It is not intended to be modified.
  • RegionLookup does not bother with the Leaf Quad node type. Since the tree is deduplicated, the space savings are minimal.
